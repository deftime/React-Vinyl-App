# About

```sh
Невеличкий браузерний додаток, який демонструє роботу Реакт.
```

```sh
Колекція музичних вінілових платівок. Є загальний каталог та користувацькі Колекція та Вибране.
Детальная інформація про платівку, демо-програвання пісні, персональні нотатки, пошук за фільтрами.
Запит і отримання інформації з серверу імітується через моки.
```

## Libraries and Packages
